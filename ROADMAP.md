# خارطة الطريق والتطوير المستقبلية (Lex Juris Pro Roadmap)

هذا الملف يحتوي على المهام والتطويرات المقترحة للمنصة، بحيث يمكن العودة إليها في أي وقت واستكمال بنائها خطوة بخطوة.

## 1. التطوير البصري وتجربة المستخدم (UI/UX Transformation)
- [x] إعادة تصميم شاملة للتطبيق ليكون متجاوباً 100% مع شاشات اللابتوب والموبايل (Responsive Design).
- [x] معالجة مشاكل القائمة الجانبية (Sidebar) وجعل حركتها انسيابية دون التأثير على تخطيط الصفحة.
- [x] ضبط شامل للألوان والثيمات (الغامق والفاتح) وإزالة أي ألوان ثابتة تعيق الرؤية في الوضع الغامق.
- [x] إضافة تأثيرات حركية (Micro-animations) وتطوير شكل الأزرار والخطوط (Cairo) لتبدو أكثر احترافية وفخامة قانونية.

## 2. تخصيص التطبيق والإعدادات المتقدمة (App Customization)
- [x] إتاحة خيار للمستخدم بإخفاء أو إظهار تبويبات القائمة الجانبية حسب الحاجة.
- [x] إمكانية تغيير مسميات التبويبات والمصطلحات داخل التطبيق لتناسب كل مستخدم (مثلاً: تغيير "طعون" إلى "دعاوى").
- [x] تخصيص شاشة البدء الافتراضية لكل مستخدم.

## 3. تيسير إدخال البيانات والذكاء الاصطناعي (Smart Data Entry)
- [x] إعداد قوائم منسدلة ديناميكية (Dynamic Dropdowns) يمكن للمستخدم تعديلها وإضافة عناصرها من الإعدادات (مثل: أسماء الدوائر، أنواع القضايا).
- [x] ميزة الإكمال التلقائي (Auto-Complete) للحقول المتكررة بناءً على الإدخالات السابقة.
- [x] توفير خيار "نسخ بيانات قضية سابقة" لتسهيل إدخال القضايا المتشابهة.

## 4. لوحة القيادة الشاملة والملفات السحابية (Case Dashboard & Cloud)
- [x] تحويل المنصة إلى (PWA) بحيث يتم تنصيبها كتطبيق مستقل على الموبايل واللابتوب بكل سهولة.
- [x] تحويل أرقام الطعون لروابط ذكية (Global Cross-Linking) تفتح ملف القضية من أي شاشة (الأجندة، التقارير).
- [x] إنشاء واجهة كاملة (Full-screen Dashboard) تعرض بيانات الطعن والخصوم بشكل عصري.
- [x] بناء محرك الملاحظات الديناميكية (Dynamic Notes Engine) لإضافة المرفقات الخاصة داخل كل طعن.
- [x] دمج أزرار الرفع المباشر على (Google Drive & OneDrive) من داخل الطعن لعرض المستندات المباشرة كـ `<iframe>`.

## 5. تطوير المذكرات القانونية والبحوث (Legal Memos & Research)
- [ ] إنشاء واجهة متقدمة ومتطورة لكتابة وإدارة المذكرات القانونية (Rich Text Editor).
- [ ] توفير نظام قوالب جاهزة (Templates) للمذكرات والصيغ القانونية.
- [ ] ربط المذكرات مباشرةً بالطعون المنظورة لسهولة الوصول إليها من بيانات الطعن.

## 6. الأرشيف الرقمي للطعون (Digital Archive)
- [ ] بناء نظام أرشيف متكامل لنقل الطعون "المنتهية" أو "المحفوظة" إليه لتخفيف الضغط عن قاعدة البيانات النشطة.
- [ ] بحث مخصص داخل الأرشيف يوفر استدعاء أي طعن قديم بكامل مرفقاته وجلساته التاريخية.
